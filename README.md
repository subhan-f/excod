# excod

A tool that scans a codebase, picks the important files, and exports them as JSON to your clipboard or a file.

## Installation

```bash
npm install -g excod
```

## Usage

```bash
excod <directory> [output.json]
```

If output.json is provided, writes to file; otherwise, prints to stdout.

## Development

```bash
npm run build
npm test
```
